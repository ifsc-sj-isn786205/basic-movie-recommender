Propriedade Float CSS - Exemplos Simples

1. Sem Float (padrão)

Este é um parágrafo normal. A imagem ocupa sua própria linha e o texto continua abaixo dela.

O texto não flui ao redor da imagem.

2. float: right

Este parágrafo tem uma imagem com float: right. O texto flui ao redor da imagem que fica à direita.

O texto continua fluindo ao redor da imagem.

3. float: left

Este parágrafo tem uma imagem com float: left. A imagem fica à esquerda e o texto flui ao redor dela.

O texto continua fluindo ao redor da imagem.

4. clear: both

Sem clear: both

Este parágrafo tem uma imagem flutuante à esquerda. O texto deve fluir ao redor da imagem que está flutuando. Este é um parágrafo mais longo para demonstrar como o texto se comporta quando não há clear: both aplicado.

Este parágrafo aparece SEM clear - ele flui ao redor da imagem flutuante acima. O texto continua fluindo ao redor da imagem porque não há nenhum elemento com clear: both para parar o fluxo.

Com clear: both

Este parágrafo tem uma imagem flutuante à esquerda. O texto deve fluir ao redor da imagem que está flutuando. Este é um parágrafo mais longo para demonstrar como o texto se comporta quando não há clear: both aplicado.

Este parágrafo aparece APÓS clear: both - ele não flui ao redor da imagem, vai para uma nova linha. O clear: both força este parágrafo a começar abaixo da imagem flutuante.

5. shape-outside: circle()

Este parágrafo demonstra o uso de shape-outside: circle(). O texto flui ao redor de uma forma circular em vez de um retângulo. A imagem tem float: left mas o texto segue o contorno circular definido pela propriedade shape-outside. Isso cria um efeito visual mais interessante onde o texto se adapta à forma da imagem.
O shape-outside permite que o texto flua ao redor de formas personalizadas, não apenas retângulos.
