MONOESPAÇADA (Monospace)

Características: Cada caractere ocupa o mesmo espaço horizontal. Essencial para código e alinhamento preciso.

Código em Monospace
function exemplo() { return "Olá, mundo!"; } // Cada caractere tem largura fixa printf("Alinhamento perfeito em colunas");

CURSIVA (Cursive)

Características: Imita escrita à mão com traços fluidos. Transmite personalidade, mas difícil para textos longos.

Uma mensagem pessoal e elegante, escrita com carinho e fluidez caligráfica.

FANTASIA (Fantasy)

Características: Fontes decorativas e temáticas. Chamativas para títulos, mas evite em textos extensos.
